StreamBro is a website for streaming games and other media. It allows you to broadcast your gameplay or media content to various people. We are made for the community.

This Repository is for the website!

Most of the stuff you see here will be front-end code as of right now.
If you have any suggestions for features or bug fixes, please let me know.

To veiw the project in your browser:

Click the green code button at the top right and then click "download zip".
For simplicity, extract the folder to your desktop folder.
You may now open the file to see the folder with the extracted content in it.
Open index.html in your browser and you should be able to see the current progress of this project and be able to navigate the pages.
